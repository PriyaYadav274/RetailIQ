# RetailIQ

## Overview

RetailIQ is an E-Commerce Business Intelligence and Predictive Analytics Platform developed using Django, MySQL, Python, and Machine Learning.

The system helps businesses analyze sales performance, customer behavior, product performance, and future sales trends through interactive dashboards and predictive analytics.

## Features

* Secure Login System
* Data Upload (CSV, Excel, JSON)
* Sales Analytics Dashboard
* Customer Analytics
* Product Analytics
* Sales & Forecasting
* PDF Report Generation
* Business KPI Monitoring
